## üìê CURVED SCREEN EFFECT WITHOUT THREE.JS

Here's the CSS 3D technique that **approximates** the curved TV look:

```css
.video-card {
  /* Create perspective space */
  transform-style: preserve-3d;
  perspective: 1000px;
  
  /* Curve effect via nested transforms */
  transform: 
    translateZ(0)
    rotateY(var(--rotation-angle))
    scale(var(--depth-scale));
  
  /* Subtle inner shadow for depth */
  box-shadow: 
    inset 0 0 20px rgba(0,0,0,0.3),
    0 10px 40px rgba(0,0,0,0.5);
  
  /* Metallic bezel */
  border: 3px solid #1a1a1a;
  border-radius: 12px;
  overflow: hidden;
}

/* The "curve" illusion */
.video-card::before {
  content: '';
  position: absolute;
  top: 0; left: 0;
  width: 100%;
  height: 100%;
  background: linear-gradient(
    90deg,
    rgba(255,255,255,0.05) 0%,
    transparent 20%,
    transparent 80%,
    rgba(0,0,0,0.1) 100%
  );
  pointer-events: none;
}
```

**The Secret:**
- Gradient overlay mimics light reflection on curved surface
- `rotateY` + perspective creates 3D positioning
- Shadow + border = premium screen bezel
- 95% as effective as true WebGL curve
